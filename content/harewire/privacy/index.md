This policy applies to all [Discord](https://discord.com) servers where the **Harewire bot** is installed.

1. **Data Collection**

  **Harewire** collects only the data necessary for its features to function. This includes:

  - **User IDs and Usernames**: To identify users for commands and roles.
  - **Server IDs**: To store settings specific to each sserver.
  - **Interaction Data**: Temporary logs of commands used to ensure the bot is working correctly.

2. **Data Storage & Third Parties**

  **Harewire** is built using [BotGhost](https://botghost.com). Therefore, all data is stored and processed on BotGhost’s secure servers and through the **Discord API**. We do not use any external databases or sell data to third parties.

3. **Sole Management**

  The bot is owned and managed exclusively by [K4LM3D](/members/k4lm3d). No other server administrators have access to the bot’s internal data or BotGhost configuration.

4. **Data Deletion**

  If you wish to have your user data deleted from the bot's memory, please contact the owner directly or leave the server where the bot is active.
